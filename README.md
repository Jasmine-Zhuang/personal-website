# Personal Website

A bilingual (EN/ZH) personal website built with Next.js and Tailwind CSS.

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- next-intl (i18n)

## Getting Started

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm run start
```

## Project Structure
- `app/` — routes and layouts
- `sections/` — page sections (Hero, Highlights, Projects, Resume)
- `components/` — shared UI components
- `content/` — profile, projects, resume data and translations
- `public/` — static assets

## i18n
Translations live in `content/messages/en.json` and `content/messages/zh.json`.
